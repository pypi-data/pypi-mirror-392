开源软件

本项目封装了一些Python开发中常用的功能，帮助你更高效地开发应用。

安装

pip install arts

功能分类

数据库

  • mysql | 访问MySQL数据库
  • mongo | 访问MongoDB数据库
  • postgre | 访问PostgreSQL数据库
  • objdb | 访问ObjDB数据库

AI

  • ollama2 | 在 windows 平台上部署 deepseek 本地版，并用 python 调用
  • openai2 | 访问ChatGPT

GUI

  • miumapp | 开发跨平台 GUI 应用

其它

  • base95 | Base95编码
